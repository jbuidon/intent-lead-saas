👥 How to set up Facebook Groups access

  1. Go to developers.facebook.com → Create App → type "Business"
  2. In your app dashboard → Add Product → add Facebook Login
  3. Go to Graph API Explorer → select your app
  4. Click Generate Access Token → add permission: groups_access_member_info
  5. Copy the token → paste it in ⚙ Settings → Facebook Token
  6. Paste your group URLs below (one per line) — you must be a member of each group
